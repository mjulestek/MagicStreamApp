## Overview

MagicStream is a modern full-stack movie streaming application used as a **hands-on DevOps and Cloud learning platform**.
The focus of this repository is on **deployment, automation, scalability, and production-ready operations** rather than feature development.

## Stack & Focus

* **Application:** React, Go (gin-gonic), MongoDB, AI recommendations
* **DevOps & Cloud:** Docker, Terraform, AWS, CI/CD, monitoring, security, and scalable architecture

## Purpose

This project showcases my **practical journey as a junior DevOps & Cloud engineer**, applying real-world tools and workflows to transform an existing system into a **cloud-ready, automated, and reliable platform**.
